# 📱 在iPhone上安装"人生地图"

## ✨ 现在已支持PWA全屏模式！

你的应用已经配置好PWA（渐进式Web应用），添加到主屏幕后将完全全屏显示，**没有Safari地址栏**。

## 📲 安装步骤

### 1. 部署到Vercel
```bash
# 推送代码到GitHub（如果还没有）
git add .
git commit -m "Add PWA support"
git push

# Vercel会自动部署
```

### 2. 在iPhone上安装

1. **打开Safari浏览器**（必须用Safari，Chrome不支持添加到主屏幕）
2. **访问你的Vercel URL**（例如：`your-app.vercel.app`）
3. **点击底部分享按钮** 📤（方框带向上箭头）
4. **向下滚动，选择"添加到主屏幕"** ➕
5. **自定义名称**（建议："人生地图"），点击**"添加"**

### 3. 完成！🎉

现在你可以：
- 从主屏幕像打开原生App一样打开应用
- **完全全屏**，没有地址栏
- 支持iPhone的安全区域（刘海屏适配）
- 数据完全本地存储，私密安全

## 🎨 PWA特性

- ✅ 全屏显示（`display: standalone`）
- ✅ 优雅的蓝色主题
- ✅ 自定义应用图标（生命之树）
- ✅ iOS状态栏样式优化
- ✅ 安全区域适配（支持刘海屏）
- ✅ 离线支持（Service Worker）

## 🔍 如何验证PWA是否正常工作

添加到主屏幕后：
1. 从主屏幕打开应用
2. 顶部应该**没有**地址栏
3. 状态栏应该与应用融为一体
4. 底部应该留出iPhone的Home indicator空间

## 💡 提示

- **必须使用Safari浏览器**进行安装
- 安装后可以在设置 > 主屏幕看到应用
- 删除：长按图标 > 移除App
- 数据不会因为删除应用而丢失（除非清除Safari浏览器数据）
